# Share

Share videos on share

## Run
  - Install [node](https://nodejs.org/en/)
  - Open a terminal
    ### Windows
    - Setup the project `./setup.bat`
    ### Unix (Mac, linux, git bash, etc.)
    - Setup the project `./setup.sh`
  - Run `npm start`
  - Open the browser
  - Goto [localhost:8080/signup](http://localhost:8080/signup)
